## 🗣 Brand Voice & Tone

- **Tone:** Friendly, clear, expert—like a seasoned service advisor  
- **Avoid:** Jargon, upsell-y language, or robotic responses  
- **Do:**  
  - Use plain English  
  - Inject warmth ("Great question!" or "Here's the scoop")  
  - Offer reassurance when answers involve costs or repairs  
- **Inline examples:**  
  - ✅ “Think of it like a processing fee.”  
  - 🚫 “This surcharge is pursuant to regulatory assessments.”  

---

## 🎨 Color, Typography, Spacing

### Primary Brand Colors
- **Blue (Primary):** `#0A64BC` — for CTA buttons, headers
- **Accent (White):** clean backgrounds, high readability
- **Soft Grays:** light borders, section dividers

### Typography
- **Font:** Inter or Söhne (clean, modern sans-serif)  
- **Headings:** bold, tight line-height  
- **Body text:** minimum 16px, max 75 characters/line  
- **Chat bubbles:** differentiate user and Archie with tone + styling

### Spacing
- Mobile-first grid  
- Generous padding around answers  
- Use white space to signal separation, not decoration

---

## 📐 Layout Best Practices

- **Mobile-first:** design for < 768px first, then scale up  
- **Full-page chat UI:** use max vertical space, hide unnecessary nav  
- **Sticky headers:** only if helpful (e.g. for CTA or topic selector)  
- **Chat interface:**
  - User messages right-aligned  
  - Archie messages left-aligned with subtle background  
  - CTA suggestions shown inline or as cards below

---

## ♿ Accessibility Must-Dos

- **Color contrast:** All text must meet WCAG AA  
- **Headings:** Semantic HTML (use `<h1>`–`<h4>` properly)  
- **Focus states:** Visible, styled outlines on buttons and inputs  
- **Labels:** Every input and button must be clearly labeled  
- **ARIA roles:** Use for complex elements (chat window, toggle menus)  
- **Motion:** Reduce motion for users with system setting enabled

## ✍️ Content Style Guide

### Headings
- Title Case for page titles (`About AskArchie.ai`)  
- Sentence case for subheadings (`How Archie is trained`)

### Lists
- Use `-` for bullets, keep items short and parallel
- Bold key terms inside list items:  
  - **Doc fee** — a dealership’s administrative cost  
  - **GAP insurance** — covers the "gap" between loan and car value

### Links
- Never use “click here”  
- Use descriptive verbs:  
  - ✅ “See car warranty options”  
  - 🚫 “Click to learn more”

### Microcopy
- Keep help text ≤ 12 words  
- Use action-first language:  
  - ✅ “Ask another question”  
  - 🚫 “You may now return to the homepage”
